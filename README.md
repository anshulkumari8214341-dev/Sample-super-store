# Sample-super-store
1.This is a report based on the analysis of the price of different apple products available on different apple stores. 
2.We are provided with excel data. 
3.We performed different operations available in excel. 
4.After that data cleaning and data visualization are performed using python and it's libraries like numpy ,pandas,matplotlib and seaborn. 
5.Then a report is prepared using the Microsoft PowerBI. Exported in the form of PDF file.
